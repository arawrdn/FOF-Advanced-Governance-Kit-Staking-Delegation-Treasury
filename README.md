# FOF-Advanced-Governance-Kit-Staking-Delegation-Treasury
A professional monorepo showcasing a robust Decentralized Autonomous Organization (DAO) structure for the Fairly Odd Fellas (FOF) NFT collection. It implements complex on-chain logic (Staking, Delegation, Voting via $VoteManager.sol$) and features a cutting-edge WalletConnect 2.0 integration using @reown/appkit and @walletconnect/core.
